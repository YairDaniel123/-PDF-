# מציג PDF מתקדם - מבוסס HTML ופייתון. לחיפוש קבצים וחיפוש בתוכן הקבצים. קל משקל מהיר נוח עם ממשק ברור וקל

בניתי לעצמי (עם AI כמובן...) תוכנה קטנה (פייתון,HTML) להצגת מסמכי PDF, וחשבתי שאני ישתף את הציבור אולי זה יועיל למישהו - וגם בכדי לשמוע ביקורות בכדי לשפר את זה
-  זה משמש אותי בגלל שיש לי הרבה קבצי PDF ואני רוצה לחפש ספר מסויים בין הקבצים שיש לי ולראות מה יש בו, בצורה מהירה וקלה. 

התוכנה יוצרת אינדקס קטן וקל משקל של כל הקבצים בתיקיות שבחרתם שיוצגו בתוכנה.

- מצב קריאה = מסך מלא

- אפשרות הוספת מסמסכים למועדפים = כפתור להצגת מועדפים בלבד
